##  FC 

FC changes the output format to FeynCalcForm. To change to InputForm use FI..

###  See also 

FeynCalcForm, FI, FeynCalcExternal, FeynCalcInternal.

###  Examples 

```mathematica
FI
{DiracGamma[5], DiracGamma[Momentum[p]]} 
 
FC
{DiracGamma[5], DiracGamma[Momentum[p]]}
```
