##  ExplicitSUNFIndex 

ExplicitSUNFIndex[ind] is a specific SU(N) index in the fundamental representation, i.e. ind is an integer..

###  See also 

SUNIndex, SUNFIndex.

###  Examples 

```mathematica
ExplicitSUNFIndex[1] 
 
SUNTF[a, 1, 2] 
 
% // FCI // StandardForm
```
