## FCGetNotebookDirectory

`FCGetNotebookDirectory[]` is a convenience function that returns the directory in which the current notebook or .m file is located. It also works when the FrontEnd is not available.

### See also

### Examples

```mathematica
FCGetNotebookDirectory[]
```
