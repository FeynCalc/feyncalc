## $LoadTARCER

`$LoadTARCER` is a legacy switch for loading TARCER. The modern way to achieve the same effect is to evaluate `$LoadAddOns={"TARCER"}` before loading FeynCalc.

### See also

[Overview](Extra/FeynCalc.md), [$LoadAddOns]($LoadAddOns.md).

### Examples

```mathematica
$LoadTARCER
```

$$\text{False}$$
