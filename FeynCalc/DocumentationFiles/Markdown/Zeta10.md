## Zeta10 

`Zeta10` denotes `Zeta[10]`.

### See also

[Zeta8](Zeta8), [Zeta6](Zeta6), [Zeta4](Zeta4), [Zeta2](Zeta2).

### Examples

```mathematica
Zeta10
```

$$\zeta (10)$$

```mathematica
N[Zeta[10]]
```

$$1.00099$$

```mathematica
SimplifyPolyLog[Pi^10]
```

```mathematica
Conjugate[Zeta10]
```
