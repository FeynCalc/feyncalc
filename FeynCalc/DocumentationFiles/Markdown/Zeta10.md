## Zeta10

`Zeta10` denotes `Zeta[10]`.

### See also

[Overview](Extra/FeynCalc.md), [Zeta8](Zeta8.md), [Zeta6](Zeta6.md), [Zeta4](Zeta4.md), [Zeta2](Zeta2.md).

```mathematica
Zeta10
```

$$\zeta (10)$$

N[Zeta[10]]
```

$$1.00099$$

SimplifyPolyLog[Pi^10]
```

$$93555 \zeta (10)$$

```mathematica
Conjugate[Zeta10]
```
