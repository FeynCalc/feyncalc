##  Li3 

Li3 is an abbreviation for the trilog function, i.e., Li3 = PolyLog[3, #]&..

###  See also 

Li2.

###  Examples 

```mathematica
Li3[x] 
 
Li3 // StandardForm 
 
D[Li3[x], x] 
 
\[Integral]Li2[x]/x \[DifferentialD]x
```
