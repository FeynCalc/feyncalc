## Zeta2

`Zeta2` denotes `Zeta[2]`.

### See also

[Overview](Extra/FeynCalc.md), [SimplifyPolyLog](SimplifyPolyLog.md).

### Examples

```mathematica
Zeta2
```

$$\zeta (2)$$

```mathematica
N[Zeta2]
```

$$1.64493$$

```mathematica
SimplifyPolyLog[Pi^2]
```

$$6 \zeta (2)$$

```mathematica
Conjugate[Zeta2]
```

$$\zeta (2)$$
