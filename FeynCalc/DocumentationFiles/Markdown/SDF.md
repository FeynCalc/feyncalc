##  SDF 

SDF[i, j] denotes the SU(N) Kronecker delta with color indices i and j in the fundamental represnetation. SDF[i,j] is transformed into SUNFDelta[SUNFIndex[i],SUNFIndex[j]] by FeynCalcInternal..

###  Examples 

```mathematica
SDF[a, b] 
 
% // FCI // StandardForm 
 
% // FCE // StandardForm
```
