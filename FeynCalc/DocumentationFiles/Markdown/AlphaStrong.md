## AlphaStrong

AlphaStrong  is a shortcut for `SMP["alpha_s"]` which represents the strong coupling constant.

The shortcut `AlphaStrong` is deprecated, please use `SMP["alpha_s"]` instead!

### See also

[Overview](Extra/FeynCalc.md), [SMP](SMP.md).

### Examples

```mathematica
AlphaStrong
% // InputForm
```

$$\alpha _s$$

```mathematica
SMP["alpha_s"]
```

```mathematica
SMP["alpha_s"]
```

$$\alpha _s$$
