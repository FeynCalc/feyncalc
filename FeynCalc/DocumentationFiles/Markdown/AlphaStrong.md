##  AlphaStrong 

AlphaStrong  is a shortcut for $\text{SMP}[\text{alpha$\_$s}]$ which represents the strong coupling constant.The shortcut AlphaStrong is deprecated, please use $text{SMP}[text{alpha$_$s}]$ instead!.

###  Examples 

```mathematica
AlphaStrong 
 
% // InputForm
```

$$\alpha _s$$

```mathematica
SMP["alpha_s"]
```
