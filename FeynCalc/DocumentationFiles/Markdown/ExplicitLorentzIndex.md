`ExplicitLorentzIndex[ind]` is an explicit Lorentz index, i.e., `ind` is an integer.

### See also

[LorentzIndex](LorentzIndex), [Pair](Pair).

### Examples

```mathematica
Pair[LorentzIndex[1], LorentzIndex[\[Mu]]]
% // StandardForm
```

$$\bar{g}^{1\mu }$$

```
(*Pair[ExplicitLorentzIndex[1], LorentzIndex[\[Mu]]]*)
```
