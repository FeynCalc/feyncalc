## $LoadPhi

`$LoadPhi` is a legacy switch for loading Phi. The modern way to achieve the same effect is to evaluate `$LoadAddOns={"Phi"}` before loading FeynCalc.

### See also

[Overview](Extra/FeynCalc.md), [\$LoadAddOns](\$LoadAddOns.md).

### Examples

```mathematica
$LoadPhi
```
