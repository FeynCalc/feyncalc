## ExplicitSUNIndex

`ExplicitSUNIndex[ind]`  is a specific $SU(N)$ index in the adjoint representation, i.e. `ind` is an integer.

### See also

[Overview](Extra/FeynCalc.md), [FCI](FCI.md), [SUNDelta](SUNDelta.md), [SUNF](SUNF.md), [SUNIndex](SUNIndex.md).

### Examples

```mathematica
ExplicitSUNIndex[1]
```
