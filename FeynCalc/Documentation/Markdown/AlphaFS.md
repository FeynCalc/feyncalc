## AlphaFS

AlphaFS  is a shortcut for `SMP["alpha_fs"]` which represents the fine-structure constant.

The shortcut `AlphaFS` is deprecated, please use `SMP["alpha_fs"]` instead!

[Overview](Extra/FeynCalc.md), [SMP](SMP.md).
