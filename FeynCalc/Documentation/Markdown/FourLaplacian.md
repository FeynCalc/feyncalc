## FourLaplacian

`FourLaplacian[exp, p, q]` is $\frac{\partial}{\partial p_{\mu }} \frac{\partial}{\partial q_{\mu }}$ applied to `exp`.

### See also

[Overview](Extra/FeynCalc.md), [FourDivergence](FourDivergence.md), [ThreeDivergence](ThreeDivergence.md).

### Examples

```mathematica
SP[q, q] 
 
FourLaplacian[%, q, q]
```
